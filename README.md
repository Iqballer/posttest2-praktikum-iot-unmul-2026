## Deskripsi
Sistem Smart House ini dirancang agar dapat mengontrol LED dari jarak jauh dengan menggunakan Bot telegram sebagai Platform IOT nya. Terdapat 4 LED yang di sistem tersebut yang dimana 3 LED dimiliki Oleh 3 User yang berbeda, LED A dapat dikontrol oleh User A, sedangkan Selain User A tidak dapat di akses Oleh user lain, Lalu terdapat 1 LED yang dimana Bebas di akses oleh semua user. Sistem ini dapat mengecek suhu ruangan dan kelembapan menggunakan sensor DHT 22. Juga terdapat sistem alarm jika terdeteksi Kebocoran gas di suatu ruangan menggunakan sensor MQ 2, alarm tersebut maka akan dikirim ke grup telegram yang didalamnya terdapat user pengguna sistem tersebut.
